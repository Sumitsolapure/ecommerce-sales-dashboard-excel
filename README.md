# 📊 Ecommerce Store Analysis Dashboard (Excel)

## 🚀 Overview
Created an interactive Excel dashboard to analyze ecommerce sales performance, customer behavior, and order status.

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Graphs  
- Slicers & Filters  

---

## 📊 Dashboard Highlights
- Gender-wise sales contribution  
- Revenue trend (monthly)  
- Best age group analysis  
- Order status breakdown  
- Top 5 cities by revenue  
- Category-wise filtering  

---

## 📈 Key Analysis
- Women contribute majority of sales (~64%)  
- Adult age group generates highest revenue  
- Most orders are successfully delivered (~92%)  
- Revenue shows variation across months  
- Top cities like Bengaluru and Hyderabad generate highest sales  

---

## 💡 Key Insights
- Targeting adult customers can increase revenue  
- Female customers are key contributors  
- Delivery success rate is strong  
- Major revenue comes from top metro cities
